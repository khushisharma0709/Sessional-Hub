# Sessional Hub

An AI-powered web application for automated question paper generation and answer evaluation. Built to simplify exam preparation, reduce manual effort, and improve academic workflow efficiency.

## Features

- Generate question papers automatically
- Evaluate answers efficiently
- User-friendly dashboard interface
- Subject and question management
- Database integration for secure storage
- Fast and scalable system
- Admin panel support

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **Database:** SQLite / MySQL
- **Libraries:** Python NLP / Automation Libraries
